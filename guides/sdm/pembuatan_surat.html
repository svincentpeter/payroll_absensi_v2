🛠️ Panduan Interaktif: Manajemen Surat & Template Surat

Halo, SDM! Berikut petunjuk lengkap di halaman Pembuatan Surat dan child page Template Surat:

  1. Form Buat Surat

    • Penerima: pilih guru/karyawan dari dropdown.
    • Judul Surat: isi judul, misal “Surat Peringatan I”.
    • Isi Surat: ketik isi lengkap di textarea.
    • Simpan: klik Buat Surat, spinner muncul, lalu form reset.
    • Validasi Bootstrap mencegah submit jika field kosong.

  2. Daftar Surat (DataTables)

    • Kolom: No, Nama Penerima, Judul, Ringkasan Isi, Tanggal Keluar, Status, Aksi.
    • Status: Terkirim atau Dibaca.
    • Sorting & pagination, export Excel/PDF/Print, responsive.

  3. Aksi Surat

    • View Detail: klik opsi “View Detail” → modal muncul dengan judul, penerima, tanggal, dan isi lengkap.
    • Hapus: klik “Hapus” → modal konfirmasi → proses AJAX hapus → daftar ter-refresh.

  4. Kelola Template Surat

    • Kelola Template Surat: klik tombol di kanan atas untuk ke template_surat.php.

  5. Child Page: Template Surat

    1. Load & Lihat Template
      • Template di-load via AJAX → ditampilkan sebagai kartu.
      • Klik kartu untuk Review → modal muncul dengan detail template.
    2. CRUD Template
      • Tambah: klik Tambah Template Surat → isi jenis, judul, isi → simpan via AJAX.
      • Edit: dari modal Review, klik Edit → form edit muncul → update via AJAX.
      • Hapus: dari modal Review, klik Hapus → konfirmasi → AJAX delete → reload daftar.
    3. Kirim Surat dari Template
      • Pilih template → modal Review → pilih penerima:
        • Perorangan: pilih anggota via modal pemilihan.
        • Semua Anggota: kirim ke semua (id_penerima=0).
      • Klik Kirim Surat → AJAX kirim → notifikasi sukses.

  6. Notifikasi & Spinner

    • Semua aksi AJAX menampilkan toast di pojok kanan atas (Durasi 3 detik).
    • Spinner muncul di button saat proses, dan di tengah layar saat load DataTables atau template.
    • Alert server (jika ada) fade otomatis setelah 3 detik.
