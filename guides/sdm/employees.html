🛠️ Panduan Interaktif: Payroll Anggota

Halo, SDM! Berikut cara menggunakan seluruh fitur di halaman Payroll Anggota dan Pengaturan Payheads:

  1. Ganti Kalender

    • Klik Ganti Kalender → pilih bulan & tahun → klik kotak bulan yang diinginkan.
    • Bulan yang sudah Final ditandai gelap dan tidak bisa dipilih.

  2. Filter Daftar Anggota

    1. Pilih Jenjang Pendidikan, Role, atau masukkan kata kunci di Pencarian.
    2. Klik Terapkan Filter untuk memuat ulang daftar kartu.
    3. Klik Reset Filter untuk mengembalikan semua setelan ke default.

  3. Menelusuri Daftar Kartu

    • Scroll untuk melihat semua kartu anggota.
    • Setiap kartu menampilkan: foto, nama, NIP, jenjang, role, status payroll, dan rapel.
    • Gunakan pagination di bawah untuk berpindah halaman.

  4. Aksi pada Setiap Kartu

    1. Detail: klik → buka modal detail data anggota.
    2. Edit: klik → ubah No. Rekening → klik Update.
    3. Payroll: klik → masuk ke halaman Pengaturan Payheads.
    4. Rekap Absensi: klik → review & edit rekap absensi.

  5. Pengaturan Payheads

    • Periksa informasi anggota dan periode payroll di panel kiri.
    • Pastikan No. Rekening, Tanggal Payroll, dan Catatan sudah diisi.
    Komponen Gaji Tersedia
    • Cari payhead melalui kotak search.
    • Klik untuk menambahkan ke daftar terpilih.
    Komponen Gaji Terpilih
    • Isi Nominal dan Keterangan untuk masing-masing baris.
    • Centang Rapel untuk menandai komponen berulang → input nominal akan dikunci.
    • Klik untuk menghapus baris jika perlu.

  6. Kenaikan Gaji Tahunan

    • Centang Aktifkan Kenaikan Gaji Tahunan untuk menampilkan field.
    • Isi Nama Kenaikan dan Nominal → akan disimpan sebagai payhead rapel.

  7. Rekap Absensi

    • Angka Total Hadir/Izin/Cuti/Tanpa Keterangan/Sakit tampil otomatis.
    • Klik tombol Review Rekap Absensi untuk melihat detail dan klik Edit jika perlu koreksi.

  8. Perhitungan dan Simpan

    1. Simpan Payroll → menyimpan data draft payheads.
    2. Proses Payroll → generate payroll & detail ke database (status draft → notifikasi ke Keuangan/Superadmin).

  9. Tips & Troubleshooting

    • Spinner berarti data sedang diproses—harap tunggu.
    • Jika tombol tidak merespons, periksa validasi form & token CSRF.
    • Pastikan semua Nominal valid (> 0) sebelum simpan atau proses.
