🛠️ Panduan Interaktif: Manajemen Data Guru/Karyawan

Halo, SDM! Berikut cara menggunakan semua fitur di halaman Manage Guru/Karyawan:

  1. Tombol Aksi Utama

    • Tambah Guru/Karyawan: klik + → isi form → klik Simpan.
    • Lihat History: klik tombol History untuk melihat perubahan data.
    • Atur Gaji Pokok: klik Atur Gaji Pokok → pilih submenu untuk strata guru/karyawan → klik Simpan.
    • Atur Salary Indeks: klik Atur Salary Indeks → pindah ke halaman Salary Indeks.
    • Atur Hari Libur: klik Atur Hari Libur → pindah ke halaman Holidays.

  2. Filter Data Guru/Karyawan

    1. Pilih Jenjang, Role, dan Status Kerja.
    2. Klik Terapkan untuk menampilkan hasil.
    3. Klik Reset untuk mengembalikan semua filter.

  3. Daftar Kartu Guru/Karyawan

    • Scroll untuk melihat semua kartu.
    • Setiap kartu menampilkan: foto, nama, NIP, masa kerja, jenjang, role, dan status.
    • Gunakan pagination di bawah untuk berpindah halaman.

  4. Aksi Per Kartu

    1. Detail: klik ikon mata → buka modal detail → baca semua informasi.
    2. Edit: klik ikon pensil → buka modal edit → ubah data → klik Update.
    3. Hapus: klik ikon tong sampah → konfirmasi → klik Hapus.

  5. Form Tambah & Edit

    • Isi semua field wajib (bertanda *).
    • Jika memilih Kontrak, akan muncul field Lama Kontrak.
    • Periksa kembali sebelum klik Simpan atau Update.

  6. Atur Gaji Pokok & Strata

    1. Atur Gaji Pokok: di modal utama → klik submenu → masukkan nominal → simpan.
    2. Atur Strata: di modal strata guru/karyawan → edit tiap jenjang & strata → klik Simpan.

  7. Tips & Troubleshooting

    • Spinner berarti data sedang diproses—tunggu hingga hilang.
    • Jika tombol tidak merespon, pastikan tidak ada error validasi pada form.
    • AJAX error? Periksa koneksi & token CSRF Anda.
