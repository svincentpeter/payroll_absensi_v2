Halo, SDM! Berikut cara menggunakan seluruh fitur di halaman Manajemen Payheads:

  1. Filter Payheads

    • Pilih Jenis Payhead (Earnings/Deductions) di dropdown.
    • Klik Terapkan untuk reload tabel sesuai filter.
    • Klik Reset untuk mengembalikan ke daftar lengkap.

  2. Tabel Daftar Payheads

    • Data ditampilkan dengan DataTables, mendukung pagination, sorting, dan search global.
    • Kolom:
      • No: Nomor urut.
      • Nama Payhead: Nama komponen gaji.
      • Jenis: Pendapatan atau Potongan.
      • Deskripsi: Keterangan singkat.
      • Nominal Tetap: Angka format Rupiah.
      • Aksi: Edit / Hapus via dropdown.
    • Loading spinner muncul di tengah saat memuat data.

  3. Export & Print

    • Klik tombol Export Excel untuk unduh .xlsx
    • Klik tombol Export PDF untuk unduh .pdf
    • Klik tombol Print untuk cetak daftar.

  4. Manage Payhead Groups

    • Klik tombol Manage Groups untuk mengatur kategori grup payheads.
    • Halaman grup memudahkan pengelompokan dan filter lanjutan.

  5. Tambah Payhead

    • Klik tombol Tambah Payhead (ikon ).
    • Isi form:
      • Nama Payhead (wajib).
      • Jenis (wajib): Earnings atau Deductions.
      • Deskripsi (wajib).
      • Nominal Tetap (wajib): angka tanpa titik/koma.
    • Klik Simpan. Validasi form wajib terpenuhi.

  6. Edit Payhead

    1. Di kolom Aksi, pilih Edit.
    2. Form Edit muncul dengan data terisi.
    3. Ubah field yang diinginkan, lalu klik Update.
    4. Nominal di-input via AutoNumeric‚Äîotomatis format ribuan.

  7. Hapus Payhead

    • Pilih Hapus di dropdown Aksi.
    • Konfirmasi di modal "Yakin ingin menghapus?".
    • Jika payhead sedang dipakai di payroll_detail, akan muncul pesan error dan tidak terhapus.

  8. Tips & Troubleshooting

    • Spinner artinya proses AJAX sedang berjalan‚Äîtunggu hingga hilang.
    • Form yang belum valid akan menampilkan feedback merah.
    • Jika terjadi error CSRF atau server, notifikasi SweetAlert2 akan muncul.
