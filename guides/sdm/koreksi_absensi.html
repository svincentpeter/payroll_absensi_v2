🛠️ Panduan Interaktif: Koreksi Absensi

Halo, SDM! Di halaman Koreksi Absensi Anda dapat melihat dan memperbaiki hasil upload fingerprint.

  1. Filter & Upload

    • Pilih Bulan: klik input month → pilih periode.
    • Pilih Departemen: klik dropdown → pilih jenjang (TK/SD/SMP/SMA/SMK).
    • Klik Tampilkan untuk memuat data.
    • Klik Upload Absensi untuk buka halaman import file Excel.

  2. Tabel Absensi

    • Scroll horizontal jika kolom banyak.
    • Gunakan tombol export (Copy / Excel / PDF / Print) di atas tabel.
    • Untuk mengedit atau menghapus baris, klik ikon ⋮ pada kolom “Aksi”.

  3. Edit Absensi

    1. Klik “Edit Absensi” → modal terbuka.
    2. Perbarui field: tanggal, jadwal, jam kerja, scan masuk/pulang, jenis absensi, dll.
    3. Klik Simpan Perubahan → tunggu spinner hilang.

  4. Hapus Absensi

    1. Klik “Hapus Absensi” → modal konfirmasi.
    2. Klik Hapus → data terhapus dan rekap otomatis ter-update.

  5. Kalender (Opsional)

    • Menampilkan tampilan bulan/minggu/hari.
    • Bisa Anda kembangkan untuk menampilkan highlight hari libur atau lembur.

  6. Tips & Troubleshooting

    • Spinner artinya sedang memuat data; tunggu hingga hilang.
    • Jika DataTables gagal load, cek token CSRF dan koneksi internet.
    • Pastikan file Excel sudah ter‑upload di halaman Upload Absensi.
