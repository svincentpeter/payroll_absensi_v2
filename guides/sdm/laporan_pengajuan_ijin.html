🛠️ Panduan Interaktif: Laporan Pengajuan Izin (SDM)

Halo, SDM! Berikut cara menggunakan semua fitur di halaman Laporan Pengajuan Izin:

  1. Notifikasi

    • Pesan sukses/danger muncul di atas konten setelah Anda aksi (Terima/Tolak/Hapus).
    • Klik ikon ✖️ di ujung alert untuk menutup notifikasi.

  2. Daftar Pengajuan Izin (Aktif)

    • Menampilkan pengajuan yang sudah Diterima Kepala Sekolah & masih Pending SDM.
    • Kolom:
      • ID, NIP, Nama
      • Judul Surat, Tanggal, Pesan
      • Tipe Izin, Lampiran
      • Status Kepala Sekolah (badge kuning)
    • Aksi:
      • Terima: ubah status SDM → “Diterima”.
      • Tolak: ubah status SDM → “Ditolak”.
    • Lampiran dibuka di tab baru; otomatis print jika di-trigger fungsi printLampiran().

  3. History Pengajuan Izin

    • Menampilkan pengajuan yang sudah diproses SDM (Diterima / Ditolak).
    • Kolom sama seperti tabel aktif, ditambah kolom Status Persetujuan SDM.
    • Aksi Hapus:
      • Hanya aktif jika status SDM ≠ “Pending”.
      • Klik Hapus → konfirmasi → data terhapus.
      • Jika masih “Pending”, tombol Hapus disable.

  4. Interaksi Tabel

    • Kedua tabel memakai DataTables (search, sort, pagination).
    • Kustom bahasa: Indonesia via CDN.
    • Jika kosong, akan muncul teks “Tidak ada…” di tabel.

  5. Validasi & Error Handling

    • Pengecekan status valid di-backend (hanya “Diterima”/“Ditolak” untuk SDM).
    • Pesan error SQL atau validasi tampil di notifikasi merah.
    • PHP error_reporting aktif (hanya untuk debugging non-produksi).
