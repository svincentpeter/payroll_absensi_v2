Halaman Rekap Payroll menampilkan ringkasan gaji untuk setiap jenjang, termasuk komponen payhead lama, grouped payhead, dan total-total penting.

  1. Filter & Tampilan Data

    • Jenjang: pilih jenjang untuk melihat rekap khusus jenjang tersebut.
    • Bulan: pilih bulan; default ke bulan berjalan.
    • Tahun: pilih tahun; default ke tahun berjalan.
    • Terapkan untuk muat ulang data, Reset untuk kembali ke default, Tampilkan Semua untuk override filter bulan.
    • Export untuk mengunduh data melalui tombol Excel/PDF/Print di toolbar tabel.

  2. Tabel Rekap Payroll

    • Kolom utama:
      • Jenjang, Total Gaji Pokok, Total Salary Index.
      • Setiap payhead lama sebagai kolom terpisah.
      • Setiap grouped payhead sebagai kolom terpisah.
      • Total Lain-lain, Total Potongan Koperasi, setiap deduction payhead, Total Gaji Bersih.
      • Aksi: tombol â€œLihat Detailâ€ untuk drill-down ke rekap per anggota jenjang.
    • Responsif dan mendukung server-side paging via AJAX (case=LoadingRekapPayroll).
    • Loading spinner muncul saat data dimuat ulang.

  3. Server-Side Processing

    • Filter, search, paging, dan ordering dikelola di backend.
    • DataTables mengirim parameter AJAX, termasuk CSRF token.
    • Audit log dicatat saat memuat rekap (AddAuditLog & LoadingRekapPayroll).

  4. Drill-Down Detail per Jenjang

    • Klik tombol Lihat Detail di kolom Aksi untuk membuka halaman detail jenjang.
    • Halaman detail menampilkan daftar karyawan beserta komponen gaji lengkap untuk jenjang tersebut.

  5. Ekspor & Cetak

    • Gunakan tombol Excel, PDF, dan Print di toolbar DataTables.
    • Ekspor hanya kolom data (kecuali kolom Aksi).
