Halaman Daftar Payroll menampilkan siapa saja yang masih berstatus draft dan siap untuk direview / difinalisasi oleh tim Keuangan.

  1. Filter & Pencarian

    • Jenjang Pendidikan: pilih jenjang untuk memfilter.
    • Role: guru, karyawan, dsb.
    • Pencarian: ketik nama atau NIP.
    • Klik Terapkan Filter untuk load ulang; Reset Filter untuk clear.

  2. Grid Card Payroll

    • Setiap kartu menampilkan:
      • Foto profil (atau placeholder).
      • Nama, NIP, Jenjang, Role.
      • Status payroll: draft, final, dsb.
      • Periode bulan/tahun payroll.
      • Review untuk membuka halaman manage-salary.php.
    • Gunakan grid responsif untuk berbagai resolusi layar.

  3. Paginasi Manual

    • Di bawah grid: angka halaman.
    • Klik nomor halaman untuk berpindah, memuat batch berikutnya.

  4. Ganti Bulan Payroll

    • Klik Ganti Kalender atau header periode.
    • Muncul modal dengan 16 bulan terakhir.
    • Klik salah satu bulan:
      • Jika masih ada yang belum final, muncul peringatan dengan daftar jenjang.
      • Konfirmasi untuk tetap beralih.
    • Setelah konfirmasi, halaman reload dengan periode baru.

  5. Keamanan & Audit

    • Semua permintaan AJAX memakai csrf_token.
    • Setiap pemilihan bulan dicatat di audit log.

  6. Tips & Troubleshooting

    • Jika kartu tidak muncul, periksa koneksi internet dan klik Terapkan Filter lagi.
    • Pastikan session belum timeout.
    • Gunakan browser terbaru untuk pengalaman terbaik.
